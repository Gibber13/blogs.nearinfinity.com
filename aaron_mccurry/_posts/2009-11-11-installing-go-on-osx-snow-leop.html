--- permalink: /blogs/aaron_mccurry/installing_go_on_osx_snow_leop.html layout: blogs title: Installing Go on OSX Snow Leopard date: 2009-11-11 14:52:23 -05:00 tags: General --- Go is a new experimental language that has been released by Google that is suppose to have nearly the speed of C while executing, without a lot of the normal headaches of C/C++ coding.  For example Go has garbage collection, dynamic types, reflections, etc. see Google Experiments With A New Language, Go for a good overview.  So after reading a few things about it, I wanted to try it out, and here is how I got it working on my 64-bit OSX machine.

It's pretty basic, all I did was follow the instructions here.

First I added:

{% highlight text %}export GOROOT=$HOME/go
export GOOS=darwin
export GOARCH=amd64
export GOBIN=$HOME/gobin
export PATH=$PATH:$GOBIN{% endhighlight %} to ~/.bash_profile

Run:

{% highlight text %}. ~/.bash_profile{% endhighlight %} Next:

{% highlight text %}sudo easy_install mercurial{% endhighlight %} This took a while for me to pull mercurial down, maybe their server was having problems.

Note: I had to install the Xcode.mpkg from the Optional Installs on my Snow Leopard DVD to get gcc-4.3 installed.

Next:

{% highlight text %}hg clone -r release https://go.googlecode.com/hg/ $GOROOT{% endhighlight %} Next:

{% highlight text %}cd $GOROOT/src
./all.bash{% endhighlight %} Note: One of the http tests goes out and touches google.com, I had to drop the firewall while building everything.  After the build and tests were complete I turned it back on.

And that's it! Time to make a hello world program.

Create hello.go:

{% highlight text %}package main

import fmt "fmt"

func main() {
   fmt.Printf("hello world!\n");
}{% endhighlight %} Compile and Link:

{% highlight text %}6g hello.go
6l hello.6{% endhighlight %} Run:

{% highlight text %}./6.out{% endhighlight %} Output:

{% highlight text %}hello world!{% endhighlight %} I am very excited about Go can't wait to write something real with it!
